# Contribuir a AI SEO Squad Controller

Gracias por tu interes en contribuir.

## Flujo recomendado

1. Haz un fork del repositorio.
2. Crea una rama desde `main` con un nombre descriptivo.
3. Realiza cambios pequenos y enfocados.
4. Abre un Pull Request con contexto tecnico claro.

## Estandares

- Mantener compatibilidad con WordPress y PHP del proyecto.
- Seguir el estilo ya existente en el codigo.
- Evitar cambios no relacionados en el mismo PR.
- Incluir pruebas manuales y pasos de validacion en la descripcion del PR.

## Commits

- Usa mensajes claros en imperativo.
- Ejemplos:
  - `Fix nonce validation in AJAX handler`
  - `Add sanitization for settings payload`

## Reporte de bugs

Incluye:

- Comportamiento esperado.
- Comportamiento actual.
- Pasos para reproducir.
- Entorno (WordPress, PHP, navegador, plugin/theme relacionado).

## Seguridad

No publiques vulnerabilidades en issues publicos. Reportalas de forma privada al mantenedor del proyecto.
