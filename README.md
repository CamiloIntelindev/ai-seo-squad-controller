# AI SEO Squad Controller

Plugin de WordPress para gestionar y coordinar funcionalidades SEO asistidas por IA.

## Estructura principal

- `ai-seo-squad-controller.php`: archivo principal del plugin.
- `includes/`: clases PHP de administración, AJAX, cliente API y manejo de datos.
- `assets/css/`: estilos para panel administrativo.
- `assets/js/`: scripts para panel administrativo.

## Instalación local

1. Copia esta carpeta dentro de `wp-content/plugins/`.
2. Activa el plugin desde el panel de WordPress.
3. Configura las opciones desde el menú de administración del plugin.
